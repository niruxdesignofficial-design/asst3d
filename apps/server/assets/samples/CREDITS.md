# Sample model credits

The GLB files in this folder are showcase/sample assets served by the mock mode
and the seeded Discover gallery. They come from the
[Khronos glTF Sample Assets](https://github.com/KhronosGroup/glTF-Sample-Assets)
repository. Per-model licenses (see each model's folder in that repo):

- Lantern, WaterBottle, Avocado, BoomBox, Corset, ToyCar — CC0 / public domain
- Fox — CC BY 4.0 (model by PixelMannen, rigging/animation by tomkranis, glTF conversion by AsoboStudio & scurest)
- CesiumMilkTruck — CC BY 4.0 (Cesium)
- MaterialsVariantsShoe — CC BY 4.0 (Shopify)
- SheenChair — CC BY 4.0 (Ronan Bigiaoui / polantis)

Replace freely with your own generated models; anything `.glb` in this folder
is picked up by the mock generator and the gallery seeds.
